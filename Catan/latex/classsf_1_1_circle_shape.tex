\doxysubsection{Detailed Description}
Specialized shape representing a circle. 

This class inherits all the functions of \doxylink{classsf_1_1_transformable}{sf\+::\+Transformable} (position, rotation, scale, bounds, ...) as well as the functions of \doxylink{classsf_1_1_shape}{sf\+::\+Shape} (outline, color, texture, ...).

Usage example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classsf_1_1_circle_shape}{sf::CircleShape}}\ circle;}
\DoxyCodeLine{circle.\mbox{\hyperlink{classsf_1_1_circle_shape_a21cdf85fc2f201e10222a241af864be0}{setRadius}}(150);}
\DoxyCodeLine{circle.\mbox{\hyperlink{classsf_1_1_shape_a5978f41ee349ac3c52942996dcb184f7}{setOutlineColor}}(\mbox{\hyperlink{classsf_1_1_color_a127dbf55db9c07d0fa8f4bfcbb97594a}{sf::Color::Red}});}
\DoxyCodeLine{circle.\mbox{\hyperlink{classsf_1_1_shape_a5ad336ad74fc1f567fce3b7e44cf87dc}{setOutlineThickness}}(5);}
\DoxyCodeLine{circle.\mbox{\hyperlink{classsf_1_1_transformable_a4dbfb1a7c80688b0b4c477d706550208}{setPosition}}(10,\ 20);}
\DoxyCodeLine{...}
\DoxyCodeLine{window.draw(circle);}

\end{DoxyCode}


Since the graphics card can\textquotesingle{}t draw perfect circles, we have to fake them with multiple triangles connected to each other. The "{}points count"{} property of \doxylink{classsf_1_1_circle_shape}{sf\+::\+Circle\+Shape} defines how many of these triangles to use, and therefore defines the quality of the circle.

The number of points can also be used for another purpose; with small numbers you can create any regular polygon shape\+: equilateral triangle, square, pentagon, hexagon, ...
