\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class sf\+::\+Rect$<$ T $>$}
Utility class for manipulating 2D axis aligned rectangles. 

A rectangle is defined by its top-\/left corner and its size. It is a very simple class defined for convenience, so its member variables (left, top, width and height) are public and can be accessed directly, just like the vector classes (\doxylink{classsf_1_1_vector2}{Vector2} and \doxylink{classsf_1_1_vector3}{Vector3}).

To keep things simple, \doxylink{classsf_1_1_rect}{sf\+::\+Rect} doesn\textquotesingle{}t define functions to emulate the properties that are not directly members (such as right, bottom, center, etc.), it rather only provides intersection functions.

\doxylink{classsf_1_1_rect}{sf\+::\+Rect} uses the usual rules for its boundaries\+: \begin{DoxyItemize}
\item The left and top edges are included in the rectangle\textquotesingle{}s area \item The right (left + width) and bottom (top + height) edges are excluded from the rectangle\textquotesingle{}s area\end{DoxyItemize}
This means that sf\+::\+Int\+Rect(0, 0, 1, 1) and sf\+::\+Int\+Rect(1, 1, 1, 1) don\textquotesingle{}t intersect.

\doxylink{classsf_1_1_rect}{sf\+::\+Rect} is a template and may be used with any numeric type, but for simplicity the instantiations used by SFML are typedef\textquotesingle{}d\+: \begin{DoxyItemize}
\item \doxylink{classsf_1_1_rect}{sf\+::\+Rect$<$int$>$} is sf\+::\+Int\+Rect \item \doxylink{classsf_1_1_rect}{sf\+::\+Rect$<$float$>$} is sf\+::\+Float\+Rect\end{DoxyItemize}
So that you don\textquotesingle{}t have to care about the template syntax.

Usage example\+:

\DoxyCodeLine{}

\DoxyCodeLine{\mbox{\hyperlink{classsf_1_1_vector2}{sf::Vector2i}}\ position(4,\ 2);}
\DoxyCodeLine{\mbox{\hyperlink{classsf_1_1_vector2}{sf::Vector2i}}\ size(18,\ 10);}
\DoxyCodeLine{\mbox{\hyperlink{classsf_1_1_rect}{sf::IntRect}}\ r2(position,\ size);}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{//\ Test\ intersections\ with\ the\ point\ (3,\ 1)}}
\DoxyCodeLine{\textcolor{keywordtype}{bool}\ b1\ =\ r1.contains(3,\ 1);\ \textcolor{comment}{//\ true}}
\DoxyCodeLine{\textcolor{keywordtype}{bool}\ b2\ =\ r2.contains(3,\ 1);\ \textcolor{comment}{//\ false}}
